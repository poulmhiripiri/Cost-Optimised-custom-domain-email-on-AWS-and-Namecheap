# Project Summary

This repository demonstrates a cost-optimised domain, DNS, email and AWS hosting foundation.

It is designed for individuals, startups, charities and small organisations that need a professional domain and branded email identity without high recurring costs.

The project highlights the author's practical experience in ISP networking, banking infrastructure, DNS operations, email routing, cloud services, account security, IAM separation and cost-aware infrastructure design.
